# 状态文档模板

本目录用于记录当前事实、最新进展、已知风险和下一步。状态文档应保持短、准、可更新，不重复背景文档和设计文档中的长期内容。

## 建议文件命名

- `project_status.md`
- `algorithm_status.md`
- `environment_status.md`
- `report_status.md`

## 模板

```markdown
# <状态主题>

## 当前结论

用简短段落说明当前已经确认的事实。

## 最新进展

- 日期：
- 变更：
- 验证：

## 当前风险

- 风险：
- 影响：
- 处理建议：

## 下一步

- [ ] 下一步 1：
- [ ] 下一步 2：
- [ ] 下一步 3：
```
